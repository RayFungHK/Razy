Crypt

The Crypt class provides simple, symmetric encryption and decryption using AES-256-CBC. It exposes only two static methods — no instantiation required.

Methods

MethodReturnDescription
Crypt::encrypt(string $text, string $key, bool $toHex = false)stringEncrypt plaintext with AES-256-CBC
Crypt::decrypt(string $encryptedText, string $key)stringDecrypt ciphertext back to plaintext

encrypt

use Razy\Crypt;

$plaintext = 'Sensitive data here';
$key       = 'my-secret-key-32chars-long!!';

// Default: returns base64-encoded string
$encrypted = Crypt::encrypt($plaintext, $key);

// With $toHex = true: returns URL-safe hex string
$hexEncrypted = Crypt::encrypt($plaintext, $key, true);

When $toHex is true, the output is a hexadecimal string safe for use in URLs, query parameters, and filenames — no base64 padding characters.

decrypt

// Decrypt base64-encoded ciphertext
$decrypted = Crypt::decrypt($encrypted, $key);
// → "Sensitive data here"

// Decrypt hex-encoded ciphertext
$decrypted = Crypt::decrypt($hexEncrypted, $key);
// → "Sensitive data here"

Example: Encrypting API Keys

use Razy\Crypt;

$apiKey    = 'sk_live_abc123def456';
$masterKey = 'app-master-encryption-key!!';

// Store encrypted key in database
$stored = Crypt::encrypt($apiKey, $masterKey);

// Retrieve and decrypt when needed
$originalKey = Crypt::decrypt($stored, $masterKey);
// → "sk_live_abc123def456"

Example: URL-Safe Tokens

use Razy\Crypt;

// Create a URL-safe encrypted token
$payload = json_encode([
    'user_id'   => 42,
    'expires'   => time() + 3600,
    'scope'     => 'read,write',
]);

$token = Crypt::encrypt($payload, $key, true);
// Safe for URLs: /verify?token=a1b2c3d4e5f6...

// Verify the token
$data = json_decode(Crypt::decrypt($token, $key), true);
if ($data['expires'] > time()) {
    // Token is still valid
}

Example: Protecting Sensitive Data

use Razy\Crypt;

// Encrypt before storing in config or database
$dbPassword  = Crypt::encrypt('db-secret-pass', $masterKey);
$smtpPassword = Crypt::encrypt('smtp-password', $masterKey);

// Decrypt at runtime
$dsn = 'mysql:host=localhost;dbname=app';
$pdo = new PDO($dsn, 'root', Crypt::decrypt($dbPassword, $masterKey));
